# mybox/ - 领航员沙箱工作区

**定位**: 安全隔离区 + 临时数据中转站

**详细规范**: 见 `docs/filesystem.md`

---

## 标准目录结构

```
mybox/
├── workspace/    # 🟢 工作区 (按任务名组织)
├── temp/         # 🟠 临时文件 (自动清理)
├── cache/        # 💾 持久化缓存
│   └── repos/    # Git 仓库缓存
└── logs/         # 📝 运行日志 (自动轮转)
```

---

## 核心规则

### 1. 安全隔离
- ✅ **读操作**: 可以读取项目任何位置
- ⚠️ **写操作**: 默认只能在 `mybox/` 内执行
- 🛡️ **系统修改**: 必须调用 `bin/` 脚本

### 2. 数据流向
```
外部文件 → mybox/temp/ (下载)
         → mybox/workspace/<task>/ (处理)
         → 任务完成后清理
```

### 3. 禁止操作
- ❌ 在 `mybox/` 根目录直接创建文件
- ❌ 在非 `mybox/` 位置写入临时文件
- ❌ 使用未规范的目录

---

## 使用场景

### 场景 1：文件处理
```
用户: "提取 PDF 里的图片"
AI: → mybox/workspace/pdf-extract/ 处理
   → 完成后清理
```

### 场景 2：任务隔离
```
用户: "爬取+分析+导出"
AI: → mybox/workspace/crawl-task/
   → 所有中间产物放此目录
   → 完成后整体清理
```

---

## 清理策略

| 目录 | 清理方式 | 时机 |
|:---|:---|:---|
| `workspace/` | 手动 | 任务完成后 |
| `temp/` | 自动/手动 | 定期 |
| `logs/` | 自动轮转 | 会话开始 |
